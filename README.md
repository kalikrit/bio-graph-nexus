# BioGraph Nexus

**Демонстрационное веб-приложение для извлечения графа знаний из текста биографии.**

На вход подаётся текст биографии, NLP-пайплайн (spaCy) выделяет сущности (люди, организации, места, даты) и связи между ними, результат сохраняется в графовую базу данных Neo4j и визуализируется на интерактивной карте (Cytoscape.js).

## Ключевые возможности

- Извлечение сущностей и связей в реальном времени
- Граф знаний в Neo4j с защитой от дубликатов
- Интерактивная визуализация с зумом и панорамированием
- Рекомендательная система: поиск похожих людей по общим связям
- Экспорт графа в формат GEXF
- Развёртывание в Docker одной командой

## Технологический стек

- **Бэкенд:** Python, FastAPI, spaCy, Neo4j
- **Фронтенд:** Vue 3 / Svelte, Cytoscape.js
- **Инфраструктура:** Docker, Redis, GitHub Actions (CI)
